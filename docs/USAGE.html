FDML — Usage

Validate, render, index, lint.

./bin/fdml validate <path> [--json] [--json-out file]
./bin/fdml validate-sch <path> [--json]
./bin/fdml validate-all <path> [--json]
./bin/fdml render corpus/valid/example-03.fdml.xml --out out/example-03.html
./bin/fdml index corpus/valid --out out/index.json
./bin/fdml lint corpus/valid [--json] [--strict]
