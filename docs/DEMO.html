FDML Demo

FDML (Folk Dance Markup Language) is a structured format and toolchain for modeling dance metadata, figures, timing, and geometry, then validating and publishing that content through a reproducible CLI + site pipeline.

3-Step Demo

1) Init

./bin/fdml init out/demo.fdml.xml \
  --title "Demo Dance" \
  --meter 3/4 \
  --tempo 96 \
  --figure-id f-demo \
  --figure-name "Basic"

2) Validate With Strict Doctor

./bin/fdml doctor out/demo.fdml.xml --strict

3) Render + Explore Search

./bin/fdml render out/demo.fdml.xml --out out/demo.html
make html
open site/search.html?q=Abdala&meter=9%2F16&genre=line

4) Export JSON

./bin/fdml export-json corpus/valid_v12/haire-mamougeh.opposites.v12.fdml.xml \
  --out site/export-json-sample.json
