FDML Demo

FDML (Folk Dance Markup Language) is a structured format and toolchain for modeling dance metadata, figures, timing, and geometry, then validating and publishing that content through a reproducible CLI + site pipeline.

3-Step Demo

1) Init

./bin/fdml init out/demo.fdml.xml \
  --title "Demo Dance" \
  --meter 3/4 \
  --tempo 96 \
  --figure-id f-demo \
  --figure-name "Basic"

v1.2 profile example:

./bin/fdml init out/demo.v12.fdml.xml \
  --title "Demo Two Lines" \
  --profile v12-twoLinesFacing

Ingest scaffold from notes text:

./bin/fdml ingest \
  --source analysis/gold/ingest/source_minimal.txt \
  --out out/ingest-minimal.fdml.xml \
  --title "Ingest Minimal" \
  --meter 4/4 \
  --tempo 112 \
  --profile v1-basic

With provenance sidecar:

./bin/fdml ingest \
  --source analysis/gold/ingest/source_minimal.txt \
  --out out/ingest-minimal.fdml.xml \
  --provenance-out out/provenance_minimal.json

2) Validate With Strict Doctor

./bin/fdml doctor out/demo.fdml.xml --strict

Need actionable guidance on failures?

./bin/fdml doctor out/demo.fdml.xml --explain

3) Render + Explore Search

./bin/fdml render out/demo.fdml.xml --out out/demo.html
make m11-validator-unified-check
make html
open site/search.html?q=Abdala&meter=9%2F16&genre=line&sourceCategory=africa

4) Export JSON

./bin/fdml export-json corpus/valid_v12/haire-mamougeh.opposites.v12.fdml.xml \
  --out site/export-json-sample.json

Generated sample JSON: site/export-json-sample.json

5) Verify End-to-End Demo Flow

make demo-flow-check
